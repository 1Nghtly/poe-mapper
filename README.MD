\# PoE Mapper Timer - Auto-Update Feature



\## For Users



\### How to Update



1\. \*\*Automatic Check\*\*: The app automatically checks for updates when you start it

2\. \*\*Manual Check\*\*: Open Settings (gear icon) â†’ Click "Check for Updates"

3\. \*\*Download\*\*: If an update is available, click "Download Update"

4\. \*\*Install\*\*: After downloading, click "Install \& Restart"



\### Update Notifications



\- \*\*Green indicator\*\*: You're on the latest version

\- \*\*Blue indicator\*\*: Update available

\- \*\*Orange indicator\*\*: Checking for updates / Downloading

\- \*\*Red indicator\*\*: Error occurred



\### What Gets Updated



Updates include:

\- Bug fixes

\- New features (like new boss tracking)

\- Performance improvements

\- UI enhancements



\### Data Safety



Your data is safe during updates:

\- Map counts are preserved

\- Settings are kept

\- Logs remain intact

\- Timer history is maintained



All your data is stored in a separate folder that doesn't get touched during updates.



\## Troubleshooting



\### "Check for Updates" button doesn't work

\- Make sure you have internet connection

\- Wait a moment and try again

\- Restart the app



\### Update downloads but won't install

\- Close the app completely (including tray icon)

\- Manually run the installer from the download notification

\- Or re-download from GitHub releases



\### Want to skip an update?

Just click "Later" when prompted. You can update anytime from Settings.
